# Fashion Wishlist Bot

Telegram-бот для создания wishlist товаров и обмена списками с друзьями.

## Технологии
- Node.js
- Express
- Telegraf (для Telegram-бота)
- Supabase (для базы данных)

## Установка и запуск
1. Клонируйте репозиторий.
2. Создайте файл `.env` в корне проекта и добавьте необходимые переменные окружения.
3. Установите зависимости:
    ```bash
    npm install
    ```
4. Запустите сервер:
    ```bash
    npm start
    ```

## Развертывание на Vercel
1. Войдите в Vercel.
2. Создайте новый проект и подключите GitHub репозиторий.
3. Добавьте переменные окружения на Vercel.
